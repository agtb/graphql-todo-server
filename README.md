# GraphQL Todo Server

Demo Todo app using a GraphQL MERN stack server.

## Installation

```bash
yarn install
```

## Usage

Create your configuration with reference to [.env](.env)

```bash
yarn start
```

## Contributing

Create your configuration in `.env.local` by copying [.env](.env)

```bash
yarn dev
```
